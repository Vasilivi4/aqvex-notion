## 🚀 Быстрый старт

### Локально
Просто открой `index.html` в браузере. Сервер не нужен.

### Деплой на Vercel (рекомендуется)
1. Зайди на [vercel.com](https://vercel.com) и создай аккаунт
2. Нажми **Add New → Project**
3. Перетащи папку с `index.html` или подключи GitHub-репозиторий
4. Нажми **Deploy** — через 30 секунд получишь публичную ссылку

### Деплой на Netlify
1. Зайди на [netlify.com](https://netlify.com)
2. Перетащи папку в раздел **Sites** (drag & drop прямо на страницу)
3. Готово — ссылка появится автоматически

### На своём сервере (Nginx)
```nginx
server {
    listen 80;
    server_name yourdomain.com;
    root /var/www/neck-mask;
    index index.html;
}
```

## 🔧 Как редактировать

### Изменить цену
Найди в `index.html` текст `149 000` и `200 000` — замени на нужные значения.

### Изменить таймер
В самом низу файла найди строку:
```js
let totalSeconds = 5 * 3600 + 47 * 60 + 22;
```
Поменяй часы / минуты / секунды на нужные.

### Изменить остаток товара
Найди `Только 9 штук!` — замени число.  
Также найди `.stock-fill { width: 18%; }` в CSS — это ширина полоски остатка (18% = почти пусто).

### Изменить УТП / офферы
Все тексты прямо в HTML — ищи по ключевым словам, например `СКИДКОЙ 50%` или `3-я в подарок`.

### Подключить форму
Замени `<a href="#" onclick="return false;">` на реальный обработчик.  
Рекомендуемые сервисы: **Formspree**, **Tally**, или свой бэкенд.
